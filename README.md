Ce script pyhton a été réalisé dans le cadre de la Licence de Physique de l'université de Bordeaux.
Il correspond à un exercice de l'UE Physique Subatomique animé par Mathias Gerbaux.

En 2026, il a été mis à jour afin de remplacer la lecture du fichier de données par expression régulière avec une lecture par pandas.

La courbe d'Aston est, en physique nucléaire, une courbe représentant l'énergie de liaison par nucléon des noyaux atomiques, en fonction de leur nombre de masse (source [Wikipedia](https://fr.wikipedia.org/wiki/Courbe_d%27Aston))

Les données proviennent Atomic Mass Data collaboration, disponible sur le site de l'Agence internationale de l'énergie atomique. Le fichier à utiliser est mass20.txt (les versions précédentes ne fonctionneront pas en raison du changement de format) : https://www-nds.iaea.org/amdc/
